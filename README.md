# Python портфолио

Добро пожаловать в моё портфолио. Здесь собраны проекты, которые я написал в процессе изучения Python.

---

## 📁 Проекты

### 🔐 Менеджер паролей
Консольное приложение для хранения паролей в JSON-файле.  
Позволяет добавлять, искать, удалять и просматривать пароли.  
[Подробнее](./password_manager/)

---

### 🎲 Генератор паролей
Генерация случайных паролей с сохранением в JSON.  
Возможность выбрать длину и количество паролей.  
[Подробнее](./prr/)

---

### 🌦️ Консольный синоптик
Программа для получения погоды через OpenWeatherMap API.  
Выводит температуру, влажность, давление, ветер и описание погоды.  
[Подробнее](./weather/)

---

### 💰 Трекер расходов (GUI)
Графическое приложение на Tkinter для учёта доходов и расходов.  
- Добавление доходов и расходов с категориями
- Просмотр всех записей в таблице
- Баланс
- Фильтрация по периоду (день, неделя, месяц)
- Топ категорий расходов
- Удаление записей по ID  
[Подробнее](./finance_tracker/)

---

## 🛠️ Используемые технологии
- Python
- JSON
- Tkinter (GUI)
- datetime
- requests (для синоптика)
- OpenWeatherMap API
